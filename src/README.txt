# Volunteer Assignment System

This project is a client-server application to assign volunteers to event services using a Genetic Algorithm.

---

## Prerequisites

- Java 17 or higher
- Gradle
- Internet connection to download dependencies (first time only)

---

## Running the Backend Server

1. Open terminal and navigate to the project root.
2. Run: `./gradlew bootRun` (Linux/Mac) or `gradlew.bat bootRun` (Windows)
3. The server will start on http://localhost:8066

---

## Running the GUI Client

1. In your IDE (e.g., IntelliJ), run the `VolunteerGUI` class.
2. Or compile and run from the command line:
   - `javac -cp . VolunteerGUI.java`
   - `java VolunteerGUI`

---

## Using the Application

1. Enter your name in the GUI.
2. Select 5 different service preferences.
3. Click "Submit Preferences" to send your data.
4. Click "Optimize Assignment" to compute volunteer assignments.
5. View the assignment results displayed in the GUI.
